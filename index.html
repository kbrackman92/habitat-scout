Habitat Scout

Apartments under €160k, ten minutes on foot from a metro that drops you at a university.

Browser tools can't legally scrape Leboncoin / SeLoger in real time (CORS + their anti-bot terms). What this scout does instead:

  1. Deep-links to each portal's Toulouse page for apartments on sale. Set price/surface in the portal's own filter UI once you land — those URL parameters change too often to bake in reliably.
  2. Maps universities → metro stations so only neighborhoods that actually serve a campus appear in the eligible list.
  3. Benchmarks any price against the local €/m² and recent comparable sales (DVF-style data).
  4. Shows demo cards illustrating how matched listings would render with the price-vs-market indicator.

For continuous, automated scraping you'd need a server-side scheduler (Python + Playwright on a VPS, or an alert service like Castorus). The benchmark data here is a snapshot — refresh against app.dvf.etalab.gouv.fr for actual notarized sales.
